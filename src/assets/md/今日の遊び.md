css transition

```css
.hoge{
  transform:rotate(360deg);
  background:#6200ee;
  transition: transform 1s 2s, background 3s 4s;
}
```

* 上記定義で360度回転を2秒後に1秒かけて行う。同時に4秒後に3秒かけて背景色が変わる。
